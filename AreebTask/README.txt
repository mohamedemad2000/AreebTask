# Automation Script for Saucedemo Website

This script automates the process of logging into the Saucedemo website, adding items to the cart, checking out an order, and logging out. It uses the Selenium library in Java.

## Installation

1. Install intellij IDE and open it.
2. Ensure you have Java installed.
3. Install the Selenium package if you haven't already.
4. Download ChromeDriver and Add it to The Project where it can be accessible.

## RUN
1. Open src Folder
2. Open test Folder
3. Open Java Folder
4. Open Testcases class
5. Run the script(testcases)
6. The script will open the Chrome browser and perform the automation steps described in the script.
7. After the script finishes, it will log out of the website and close the browser automatically.
